# 思维工具详解

本目录包含五大思维工具的详细说明文档。每个工具都有独立的文件，方便深入学习和参考。

## 工具列表

### 01-first-principles.md
**第一性原理** - 拆解到不可再分的基本真理
- 适用场景：创新突破、打破惯性思维、资源受限
- 核心方法：列出假设 → 质疑假设 → 保留事实 → 重新构建

### 02-systems-thinking.md
**系统思维（冰山模型）** - 看到表象下的结构
- 适用场景：问题反复出现、治标不治本、寻找根本原因
- 核心方法：表象层 → 模式层 → 结构层 → 心智模式层

### 03-inversion.md
**逆向思维** - 从反面思考问题
- 适用场景：风险识别、避免失败、突破思维定式
- 核心方法：定义成功 → 反向定义失败 → 列出失败因素 → 逐一避免

### 04-mental-models.md
**多元思维格栅** - 从多学科视角分析
- 适用场景：复杂问题、需要多角度验证、创新解决方案
- 核心方法：选择3-5个学科视角 → 交叉分析 → 寻找交叉点 → 综合洞察

### 05-critical-thinking.md
**批判思考（Paul-Elder框架）** - 质疑假设和验证逻辑
- 适用场景：重大决策、评估建议、验证分析
- 核心方法：质疑事实、证据、假设、视角、推论

## 如何使用

1. **学习阶段**：逐个阅读每个工具的详细说明
2. **实践阶段**：在实际问题中尝试使用单个工具
3. **组合阶段**：根据问题类型组合使用多个工具
4. **精通阶段**：自动选择最合适的工具组合

## 工具选择建议

| 问题类型 | 推荐工具 |
|---------|---------|
| 创新突破 | 第一性原理 + 多元思维格栅 |
| 根本原因 | 系统思维 + 批判思考 |
| 风险识别 | 逆向思维 + 批判思考 |
| 战略决策 | 全部工具组合 |

详细的调用方法请参考 [03-user-guides/tool-invocation.md](../03-user-guides/tool-invocation.md)。
