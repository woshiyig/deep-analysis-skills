# 快速开始

5分钟了解如何使用深度分析技能。

## 什么是深度分析技能？

一个整合了五大思维工具的系统性分析框架：
1. 第一性原理 - 拆解到基本真理
2. 系统思维 - 看到深层结构
3. 逆向思维 - 从反面思考
4. 多元思维格栅 - 多学科视角
5. 批判思考 - 质疑和验证

## 如何使用？

### 最简单的方式

直接说出你的问题：
```
"帮我深度分析..."
"用系统性思维分析..."
"我需要全面分析..."
```

### 示例

**问题**："我是否应该换工作？"

**调用**：
```
"帮我深度分析是否应该换工作"
```

**你会得到**：
- 执行摘要（3句话核心洞察）
- 问题拆解（假设、事实、推导）
- 系统性分析（冰山模型四层）
- 多元视角（心理学、经济学、职业发展理论）
- 行动建议（按优先级分类）
- 风险地图和缓解策略

## 三种分析深度

### 快速模式（5-10分钟）
```
"快速分析这个问题"
```
跳过外部研究，只用1-2个思维工具。

### 标准模式（20-30分钟）
```
"帮我分析..."（默认）
```
完整的对话澄清 + 系统分析，使用2-3个思维工具。

### 深度模式（30-60分钟）
```
"深度研究这个问题"
```
启动外部研究代理，使用全部思维工具，包含案例分析和延伸阅读。

## 适用场景

✅ **适合使用**：
- 复杂问题（涉及多个维度）
- 战略决策（需要全面评估）
- 系统性问题（问题反复出现）
- 跨学科问题（需要多视角）

❌ **不适合使用**：
- 简单问题（答案明确）
- 技术问题（纯技术实现）
- 紧急问题（需要立即行动）
