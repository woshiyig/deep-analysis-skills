# 工作流程

本目录详细说明深度分析的三个阶段工作流程。

## 三阶段流程

### 阶段1：对话式澄清（5-10分钟）
通过苏格拉底式提问，帮你澄清问题本质。
详见：[stage1-clarification.md](./stage1-clarification.md)

### 阶段2：系统性研究（20-30分钟）
并行启动研究代理，进行多维度分析。
详见：[stage2-research.md](./stage2-research.md)

### 阶段3：结构化输出（10-15分钟）
生成完整的分析报告和行动建议。
详见：[stage3-output.md](./stage3-output.md)

## 流程图

```
用户提问
    ↓
阶段1：对话式澄清
    ├─ 事实层：具体发生了什么？
    ├─ 假设层：你认为原因是什么？
    ├─ 目标层：理想状态是什么？
    ├─ 约束层：有哪些限制条件？
    └─ 视角层：其他人会怎么看？
    ↓
阶段2：系统性研究
    ├─ 研究代理1：历史根源和结构性原因
    ├─ 研究代理2：影响机制和传导路径
    └─ 研究代理3：应对策略和行动建议
    ↓
阶段3：结构化输出
    ├─ 执行摘要
    ├─ 问题拆解（第一性原理）
    ├─ 系统性分析（冰山模型）
    ├─ 多元视角（思维格栅）
    ├─ 行动建议
    └─ 风险地图
    ↓
完整分析报告
```

## 灵活控制

你可以随时：
- **跳过阶段1**：说"全面分析"或"直接开始分析"
- **加速阶段2**：说"跳过研究"或"基于现有信息分析"
- **简化阶段3**：说"只要核心洞察"或"给我简短版本"

## 时间分配

| 模式 | 阶段1 | 阶段2 | 阶段3 | 总计 |
|------|-------|-------|-------|------|
| 快速 | 跳过 | 5分钟 | 5分钟 | 10分钟 |
| 标准 | 5分钟 | 15分钟 | 10分钟 | 30分钟 |
| 深度 | 10分钟 | 30分钟 | 15分钟 | 55分钟 |

详细说明请查看各阶段的文档。
