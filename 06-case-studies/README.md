# 案例库

本目录包含不同领域的深度分析案例，展示如何在实际问题中应用整合思维框架。

## 案例分类

### business/ - 商业案例
商业战略、市场分析、投资决策等商业领域的案例。

**现有案例**：
- 电力现货市场与虚拟电厂分析.md - 能源行业的市场分析和投资机会

### management/ - 管理案例
团队管理、领导力、组织发展等管理领域的案例。

### product/ - 产品案例
产品方向、用户需求、功能设计等产品领域的案例。

### personal/ - 个人成长案例
职业发展、学习方法、习惯养成等个人成长领域的案例。

## 如何使用案例

1. **学习参考**：查看类似问题的分析思路和方法
2. **模板借鉴**：参考案例的结构和章节设置
3. **工具应用**：学习如何在实际问题中应用思维工具
4. **质量标准**：了解高质量分析报告的标准

## 案例结构

每个案例通常包含：
1. **问题描述**：清晰定义要分析的问题
2. **使用工具**：说明使用了哪些思维工具
3. **分析过程**：展示完整的分析过程
4. **核心洞察**：提炼关键发现
5. **行动方案**：提供可执行的建议
6. **验证方法**：说明如何验证分析结果

## 案例质量标准

优质案例应该：
- ✅ 问题定义清晰
- ✅ 分析过程完整
- ✅ 使用多个思维工具
- ✅ 洞察深刻且可操作
- ✅ 结构清晰易读
- ✅ 包含验证方法

## 学习路径

1. **初学者**：从简单案例开始，了解基本分析流程
2. **进阶者**：学习复杂案例，掌握工具组合使用
3. **高级者**：分析案例背后的思维模式，形成自己的分析风格

查看具体案例请进入各分类目录。
