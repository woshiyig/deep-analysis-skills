# 核心技能定义

本目录包含深度分析技能的核心定义文件。

## 文件说明

### deep-analysis.md
主技能文件，定义了深度分析技能的基本结构、触发方式和工作流程。适合快速了解如何使用这个技能。

### integrated-thinking-framework.md
完整的思维框架文档，详细说明了五大思维工具的使用方法、适用场景和实战案例。适合深入学习每个思维工具的细节。

## 五大思维工具

1. **第一性原理** - 拆解到不可再分的基本真理
2. **系统思维（冰山模型）** - 看到表象下的结构和深层原因
3. **逆向思维** - 从反面思考，识别风险和陷阱
4. **多元思维格栅** - 从多学科视角交叉分析
5. **批判思考（Paul-Elder框架）** - 质疑假设，验证逻辑
