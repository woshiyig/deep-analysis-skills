# 深度分析技能

基于整合思维框架（第一性原理、系统思维、逆向思维、多元思维格栅、批判思考）的系统性分析技能。

## 何时使用

- 遇到复杂的、多维度的问题
- 需要从多个学科视角交叉分析
- 需要找到问题的根本原因和系统性解决方案
- 用户明确要求"深度分析"或"系统性分析"

## 工作流程

### 阶段1：对话式澄清（5-10分钟）
通过苏格拉底式提问澄清问题：
1. 事实层：具体发生了什么？
2. 假设层：你认为原因是什么？
3. 目标层：理想状态是什么？
4. 约束层：有哪些限制条件？
5. 视角层：其他人会怎么看？

### 阶段2：系统性研究（20-30分钟）
并行启动多个研究代理：
1. 历史根源和结构性原因分析
2. 影响机制和传导路径分析
3. 应对策略和行动建议研究

### 阶段3：结构化输出（10-15分钟）
生成完整的分析报告，包含：
- 执行摘要
- 问题拆解（第一性原理）
- 系统性分析（冰山模型）
- 多元视角（思维格栅）
- 具体影响分析
- 行动建议
- 风险地图

## 分析框架

### 🔷 第一性原理
拆解到不可再分的基本真理：
1. 列出所有假设
2. 质疑每个假设
3. 保留基本事实
4. 重新构建解决方案

### 🔷 系统思维（冰山模型）
看到表象下的结构：
- 表象层：可见事件
- 模式层：重复模式
- 结构层：系统结构
- 心智模式层：深层信念

### 🔷 逆向思维
从反面思考：
- 如果想让事情失败，我会怎么做？
- 最坏情况是什么？
- 如何避免这些情况？

### 🔷 多元思维格栅
从多学科视角分析：
- 心理学、经济学、生物学
- 物理学、数学、历史学
- 寻找交叉点和矛盾点

### 🔷 批判思考（Paul-Elder框架）
五个维度质疑：
- 事实：这是事实还是观点？
- 证据：证据可靠吗？
- 假设：隐含了什么前提？
- 视角：其他人会怎么看？
- 推论：逻辑链条完整吗？

## 输出格式

生成完整的 Markdown 分析报告，包含：
1. 执行摘要（3句话）
2. 问题拆解（假设、事实、推导）
3. 系统性分析（冰山模型四层）
4. 多元视角（3-5个学科视角）
5. 具体影响分析
6. 行动建议（按优先级和风险等级）
7. 验证方法和成功指标
8. 风险地图和缓解策略
9. 延伸阅读

## 使用示例

**触发方式**：
- "帮我深度分析美伊冲突的影响"
- "用系统性思维分析这个问题"
- "我需要全面分析一下..."

**分析深度控制**：
- 快速模式："快速分析"（5-10分钟）
- 标准模式：默认（20-30分钟）
- 深度模式："深度研究"（30-60分钟）
