# 模板库

本目录包含不同场景的分析报告模板，可以直接使用或根据需要修改。

## 模板列表

### analysis-report-template.md
通用分析报告模板，适用于大多数复杂问题的深度分析。

### decision-making-template.md
决策分析模板，适用于需要在多个选项中做出选择的场景。

### problem-solving-template.md
问题解决模板，适用于需要找到根本原因并提出解决方案的场景。

### strategic-planning-template.md
战略规划模板，适用于长期规划和战略决策。

## 如何使用模板

1. **选择合适的模板**：根据你的问题类型选择最接近的模板
2. **复制模板内容**：将模板内容复制到新文件
3. **填充内容**：根据你的具体问题填充各个章节
4. **调整结构**：根据需要增删章节或调整顺序

## 模板适用场景

| 模板 | 适用场景 | 关键章节 |
|------|---------|---------|
| 通用分析报告 | 复杂问题、多维度分析 | 问题拆解、系统分析、多元视角 |
| 决策分析 | 选择困难、权衡利弊 | 选项对比、风险评估、决策矩阵 |
| 问题解决 | 问题反复出现、治标不治本 | 根本原因、解决方案、验证方法 |
| 战略规划 | 长期规划、方向选择 | 环境分析、战略选择、实施路径 |

## 自定义模板

你可以基于现有模板创建自己的模板：
1. 选择最接近的模板作为基础
2. 根据你的行业或领域调整章节
3. 添加特定的分析工具或框架
4. 保存为新的模板文件

## 模板与思维工具的结合

每个模板都可以结合五大思维工具使用：
- **第一性原理**：用于问题拆解章节
- **系统思维**：用于根本原因分析章节
- **逆向思维**：用于风险识别章节
- **多元思维格栅**：用于多维度分析章节
- **批判思考**：用于假设验证章节

详细的工具使用方法请参考 [02-thinking-tools](../02-thinking-tools/) 目录。
