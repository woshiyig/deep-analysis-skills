## 🔷 五大思维工具

### 1. 第一性原理
拆解到不可再分的基本真理，从根本出发重新构建解决方案。

**适用场景**：创新突破、打破惯性思维、资源受限

### 2. 系统思维（冰山模型）
看到表象下的结构、模式和心智模式，找到根本原因。

**适用场景**：问题反复出现、治标不治本、寻找根本原因

### 3. 逆向思维
从反面思考问题，识别风险和陷阱。

**适用场景**：风险识别、避免失败、突破思维定式

### 4. 多元思维格栅
从多学科视角（心理学、经济学、生物学等）交叉分析。

**适用场景**：复杂问题、需要多角度验证、创新解决方案

### 5. 批判思考（Paul-Elder框架）
质疑假设，验证逻辑，从事实、证据、假设、视角、推论五个维度分析。

**适用场景**：重大决策、评估建议、验证分析

详细说明请查看 [思维工具详解](02-thinking-tools/)。

## 🎯 适用场景

### ✅ 适合使用
- 复杂问题（涉及多个维度、多方利益相关者）
- 战略决策（需要全面评估风险和机会）
- 系统性问题（问题反复出现，需要找根本原因）
- 跨学科问题（需要从多个专业视角分析）

### ❌ 不适合使用
- 简单问题（答案明确，不需要深度分析）
- 技术问题（纯技术实现，不涉及复杂决策）
- 紧急问题（需要立即行动，没时间深度分析）

## 📊 分析深度

| 模式 | 时长 | 特点 |
|------|------|------|
| 快速模式 | 5-10分钟 | 跳过外部研究，使用1-2个工具 |
| 标准模式 | 20-30分钟 | 完整流程，使用2-3个工具（默认） |
| 深度模式 | 30-60分钟 | 启动研究代理，使用全部工具 |
