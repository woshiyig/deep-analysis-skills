# 深度分析技能 (Deep Analysis Skills)

基于整合思维框架的系统性深度分析技能，适用于 Claude Code 和其他 AI 助手。

## 项目简介

这是一个强大的深度分析技能包，整合了五大经典思维工具：

1. **第一性原理** - 拆解到不可再分的基本真理
2. **系统思维（冰山模型）** - 看到表象下的结构和模式
3. **逆向思维** - 从反面思考问题，识别风险
4. **多元思维格栅** - 从多学科视角交叉分析
5. **批判思考（Paul-Elder框架）** - 五个维度质疑假设和推论

## 功能特性

✨ **混合模式工作流**
- 对话式澄清（5-10分钟）
- 系统性研究（20-30分钟）
- 结构化输出（完整分析报告）

🎯 **灵活的分析深度**
- 快速模式：5-10分钟
- 标准模式：20-30分钟
- 深度模式：30-60分钟

📊 **完整的输出格式**
- 执行摘要
- 问题拆解（第一性原理）
- 系统性分析（冰山模型）
- 多元视角（思维格栅）
- 行动建议
- 风险地图

## 快速开始

### 安装

1. 将技能文件复制到 Claude Code 技能目录：

```bash
cp skills/deep-analysis.md ~/.claude/skills/
cp skills/integrated-thinking-framework.md ~/.claude/skills/
```

2. 重启 Claude Code 或重新加载技能。

### 使用示例

**方式1：自动触发**
```
"帮我深度分析是否应该换工作"
"用系统性思维分析团队效率低下的问题"
```

**方式2：指定思维工具**
```
"用第一性原理分析这个商业模式"
"用系统思维的冰山模型分析..."
"从多元视角分析这个决策"
```

**方式3：控制分析深度**
```
"快速分析这个问题"（5-10分钟）
"深度研究这个问题"（30-60分钟）
```

## 适用场景

### ✅ 适合使用
- 复杂问题（涉及多个维度、多方利益相关者）
- 战略决策（需要全面评估风险和机会）
- 系统性问题（问题反复出现，需要找根本原因）
- 跨学科问题（需要从多个专业视角分析）

### ❌ 不适合使用
- 简单问题（答案明确，不需要深度分析）
- 技术问题（纯技术实现，不涉及复杂决策）
- 紧急问题（需要立即行动，没时间深度分析）

## 实战案例

查看 `examples/电力现货市场与虚拟电厂分析.md` 了解完整的分析报告示例。

**问题**：为什么美伊战争总是不能结束？对世界金融秩序有何影响？普通人应该怎么办？

**使用工具**：
- 第一性原理：拆解到身份对立、地缘博弈、能源体制竞争
- 系统思维：分析冲突的自我强化循环
- 多元视角：地缘政治学、经济学、金融学、历史学
- 逆向思维：最坏情况分析

**输出**：12,000字完整分析报告，包含具体的应对策略
